bayes_opt.event¶

OptimizationEvent¶

enum bayes_opt.event.OptimizationEvent(value)[source]¶
Overview:

Events during optimization.

Member Type

int

Valid values are as follows:

START = <OptimizationEvent.START: 1>¶

Start event.

STEP = <OptimizationEvent.STEP: 2>¶

Step event.

SKIP = <OptimizationEvent.SKIP: 3>¶

Skipped event.

END = <OptimizationEvent.END: 4>¶

End event.

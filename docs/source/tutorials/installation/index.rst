Installation
===================

``hbayes`` is currently hosted on PyPI. It required python >= 3.6.

You can simply install ``hbayes`` from PyPI with the following command:

.. code:: shell

    pip install hbayes

You can also install with the newest version through GitHub:

.. code:: shell

    pip install -U git+https://github.com/hansbug/BayesianOptimization.git@main

of ``hbayes`` should be shown.
